## 📖 About The Project

Managing a high-volume restaurant requires a system that refuses to lag. **Gecko RMS** is a comprehensive, real-time operating system built to handle the chaos of the hospitality industry. 

By unifying the front-of-house (waitstaff), back-of-house (kitchen), and management into a single, instantly-syncing ecosystem, it eliminates missing tickets, speeds up table turnover, and provides executives with crystal-clear financial insights.

## ✨ Key Features

* **Instant Sync Engine:** Changes propagate across all devices (Admin, POS, Kitchen) in under 200ms. No refreshing required.
* **Khata (Credit) Ledger:** Built-in digital credit book to track loyal customer tabs, advances, and due clearances.
* **Smart Billing:** Auto-calculate taxes, apply dynamic discounts, and generate thermal-printer-ready receipts.
* **Role-Based Access Control (RBAC):** Strict permission boundaries ensuring cashiers, chefs, and admins only access their specific domains.
* **Dynamic Table Routing:** Interactive floor plan to manage occupied, vacant, and billing-pending tables at a glance.

## 🏗️ System Architecture

Gecko RMS relies on a modern, highly scalable tech stack:

* **Frontend:** Next.js (App Router), React 19, Tailwind CSS, Framer Motion (for hardware-accelerated 120fps UI animations).
* **Backend & Auth:** Node.js, Next.js Server Actions, Supabase (PostgreSQL, Row Level Security).
* **Infrastructure:** Self-Hosted Ubuntu VPS, CloudPanel, Nginx Reverse Proxy.
